The Salt Lake Modmail bot acts as a bridge. It creates a Private Thread for every user.

  • The Rule: NEVER reply in the main channel or ping the user.
  • Why? The user cannot see the main channel. Pinging them breaks the immersion and creates clutter.
  • How to Reply: Just type in the thread. The bot mirrors your message to their DM automatically.

?note [text]
Creates a gray embed visible only to staff. Use this for logging warnings or discussing the user privately within the ticket.

?id
Extracts the User ID. Always use ID for bans, not usernames.

?close
Closes the ticket. Use this only when the issue is fully resolved.

HIGH Harassment, Slurs in Chat, Ban Appeals.

NORMAL General questions, "How do I get roles?", Feedback.
